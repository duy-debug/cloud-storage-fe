# FormTextarea

Textarea với label, placeholder, rows, bắt buộc, lỗi và trợ giúp.

## Mục đích & trường hợp xử lý

- Nhập nội dung văn bản dài (mô tả, ghi chú).

## Props

| Tên | Kiểu | Mặc định | Mô tả |
| --- | --- | --- | --- |
| label | `string` | - | Nhãn phía trên textarea |
| placeholder | `string` | `''` | Gợi ý nội dung |
| required | `boolean` | `false` | Bắt buộc; thêm dấu `*` và thuộc tính `required` |
| error | `string \| null` | - | Thông báo lỗi dưới trường |
| help | `string \| null` | - | Gợi ý/hướng dẫn dưới trường |
| rows | `number` | `4` | Số hàng hiển thị |
| className | `string` | - | Bổ sung class Tailwind |
| ...rest | `Omit&lt;React.TextareaHTMLAttributes&lt;HTMLTextAreaElement&gt;, 'placeholder' &#124; 'rows'&gt;` | - | Thuộc tính textarea gốc |

## Cách dùng
```tsx
<FormTextarea label="Ghi chú" placeholder="Nhập ghi chú..." rows={6} />
```

## Lưu ý

- Khi có `error`, textarea có `aria-invalid` và `aria-describedby` tới thông báo lỗi.
- Dùng `rows` để đặt chiều cao mặc định; thành phần đã `resize-none` để cố định.
