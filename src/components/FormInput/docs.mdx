# FormInput

Input văn bản với label, placeholder, bắt buộc, lỗi và trợ giúp.

## Mục đích & trường hợp xử lý

- Thu thập chuỗi văn bản (email, password, số, v.v.) với các trạng thái.

## Props

| Tên | Kiểu | Mặc định | Mô tả |
| --- | --- | --- | --- |
| label | `string` | - | Nhãn hiển thị phía trên input |
| type | `React.HTMLInputTypeAttribute` | `'text'` | Kiểu input (`text`, `email`, `password`, ...) |
| placeholder | `string` | `''` | Gợi ý nội dung |
| required | `boolean` | `false` | Bắt buộc; thêm dấu `*` và thuộc tính `required` |
| error | `string \| null` | - | Thông báo lỗi dưới trường |
| help | `string \| null` | - | Gợi ý/hướng dẫn dưới trường |
| className | `string` | - | Bổ sung class Tailwind |
| ...rest | `Omit&lt;React.InputHTMLAttributes&lt;HTMLInputElement&gt;, 'type' &#124; 'placeholder'&gt;` | - | Thuộc tính input gốc |

## Cách dùng
```tsx
<FormInput label="Email" type="email" placeholder="you@example.com" required />
```

## Lưu ý

- Khi có `error`, input có `aria-invalid` và `aria-describedby` tới thông báo lỗi.
- Có thể truyền `id` để quản lý liên kết `label` ↔ `input`.
