# FormButton

Nút dùng trong form với các biến thể màu và kích thước.

## Mục đích & trường hợp xử lý

- Hành động trong form (submit, reset hoặc các thao tác phụ trợ).
- Hỗ trợ nhiều `variant` và `size`, kèm trạng thái `disabled`.

## Props

| Tên       | Kiểu                                                             | Mặc định    | Mô tả                  |
| --------- | ---------------------------------------------------------------- | ----------- | ---------------------- |
| type      | `'button' \| 'submit' \| 'reset'`                                | `'button'`  | Kiểu nút trong form    |
| variant   | `'primary' \| 'secondary' \| 'danger' \| 'success' \| 'warning'` | `'primary'` | Biến thể màu           |
| size      | `'sm' \| 'md' \| 'lg' \| 'xl'`                                   | `'md'`      | Kích thước nút         |
| disabled  | `boolean`                                                        | `false`     | Vô hiệu hóa nút        |
| className | `string`                                                         | -           | Bổ sung class Tailwind |
| ...rest   | `React.ButtonHTMLAttributes<HTMLButtonElement>`                  | -           | Thuộc tính button gốc  |

## Cách dùng

```tsx
<FormButton type="submit" variant="primary" size="md">
  Lưu
</FormButton>
```

## Lưu ý

- `disabled` sẽ thêm `opacity-50 cursor-not-allowed` và set thuộc tính `disabled`.
- `type` mặc định là `'button'`; đặt `'submit'` khi dùng trong `<form>` để gửi dữ liệu.
