# AccountDropdown

```tsx
import AccountDropdown from '@/components/Navbar/AccountDropdown/AccountDropdown'
import { AccountDropdownDemoControlled } from '@/components/Navbar/AccountDropdown/accountdropdown.demos'

export default function Example() {
  return (
    <div className="space-y-6">
      <div className="flex justify-end p-6">
        <AccountDropdown userName="Jane Doe" userEmail="jane@example.com" />
      </div>
      <AccountDropdownDemoControlled />
    </div>
  )
}
```

## Props

- **userName**: string (default: "User")
- **userEmail**: string (default: "user@example.com")
- **userAvatar**: string | null (default: null)
- **settingsHref**: string (default: "#")
- **onLogout**: `() => Promise<void> | void`
- **open**: boolean (controlled)
- **defaultOpen**: boolean (uncontrolled default)
- **onOpenChange**: (open: boolean) => void

## Notes

- Icons use Heroicons directly (`@heroicons/react`).
- Click outside or press Escape to close.
- Logout shows a confirmation `Dialog`. Provide `onLogout` to handle the action.
